# ❤️ Heart — AR Anatomical Model + Voice AI Avatar

## About

An AR experience that places a 3D anatomical heart model directly in real space, paired with a **voice-driven AI avatar**. Speak a question about the heart, and the avatar answers out loud — turning anatomy learning into a conversation instead of a slide.

## ✨ Features

- 🫀 Real-time AR rendering of a 3D anatomical heart model
- 🎙️ Voice-based interaction — speak your question, hear the answer
- 🤖 AI-driven Q&A about heart anatomy and function
- 📱 Designed for mobile/tablet AR
